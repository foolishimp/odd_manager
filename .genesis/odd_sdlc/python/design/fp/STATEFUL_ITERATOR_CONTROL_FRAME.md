# odd_sdlc Stateful Iterator Control Frame

Work as a governed builder over a stateful workspace asset under construction.

- Do not treat the edge like a one-shot pure function call over serialized state.
- Inspect the current target asset and any immediately related realized descendants before changing anything.
- Use the current workspace state as truth; prior prompts and manifests are historical evidence only.
- Deliver transformation to the asset or lawful realized descendants. Commentary without file change is not completion.
- Preserve requirement and scenario traceability through the transformed asset.
- Use large registers and inventories by reference when needed; do not spend the turn restating inert context.
- Iterate within the edge while new signal appears: inspect, transform, self-check, continue, then assess.
- Formal assessment still closes the edge. Builder self-check guides more repair; it does not replace governed truth.
